Prismx es un proyecto que busca promover la inclusion del a comunidad lgbt a traves de la informacion, las historias personales y la investigacion pueden complementarse para proporcionar una mejor perepectiva de las problematicas que afronta la comunidad en su vida.
Esto esta contruido mediante una pagina web alojada en GitHub pages, con una base de datos contruida en oracle cloud que se conecta mediante el uso de una API para la realización de consultas. También se vincula a recursos audiovisuales alojados en redes sociales como youtube y tiktok usando sus propias herramientas para inserción web , con lo que buscamos lograr un mejor alcance.
